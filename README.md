# node-vampinto-bot

Envia a foto do Vampeta pelado quando algum usuário do Twitter usa hashtag específica.

## Instalação

Extraia ```"imagem.rar"``` em ```"src/"```:

Em seguida, execute o comando de instalação do seu gerenciador de pacotes:

```npm i``` ou ```yarn```
